We started learning with basic computer studies where I learnt parts and uses of computers and many others.

After that, I learnt about version control system(Git) and how it is used. It is used for saving files, tracking changes in files and collaborating with other developers on projects.

I learnt HTML which is the structure of a webpage. I learnt the tags used, how to use them and when to use them.

Next, we moved to CSS(Cascading Style Sheet). Just as HTML is for building the structure, CSS is used for styling this structure. It gives it the design needed to make it beautiful.

Padding refers to the space around a content. In a box model, it is used to fix the space between the content and the border.

Margin is the space around a content outside any defined borders. It is used to create extra space around an element

Borders are box-like lines around a content. It is used to separate or draw attention to a content on a webpage. It comes between the padding and margin.
